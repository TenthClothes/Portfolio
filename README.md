## 1. Credit Card Fraud Detection (XGBoost)

This project focuses on building a classification model to detect fraudulent credit card transactions.

### Objective
Predict whether a transaction is fraudulent or legitimate based on transaction features.

### Approach
- Data preprocessing and feature handling  
- Train-test split for model evaluation  
- Model training using XGBoost Classifier  
- Performance evaluation using metrics such as accuracy, precision, recall, and F1-score  

### Key Techniques
- Gradient Boosting (XGBoost)  
- Classification modeling  
- Feature engineering  
- Model evaluation metrics

## 2. Hard-Coded Linear Regression Model

This project implements linear regression from scratch without using ML libraries like Scikit-learn.

### Objective
Fit a best-fit line to randomly generated data points.

### Approach
- Randomly generate 10 (x, y) coordinate points  
- Iteratively adjust parameters to minimize loss  
- Use Mean Squared Error (MSE) as the loss function  

### Key Techniques
- Gradient descent (manual implementation)  
- Loss function optimization  
- Basic regression fundamentals  
- Parameter tuning (weights & bias updates)
